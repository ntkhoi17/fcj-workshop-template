Attended Events

During my internship, I participated in 3 events, each being a memorable experience with new, interesting, and useful knowledge, along with wonderful gifts and moments.

Event 1 - FCAJ Community Day (May 23)

  • Event Name: FCAJ Community Day
  • Event Time: 9:00 AM – 12:00 PM, Saturday, May 23, 2026
  • Location: 26th Floor, Bitexco Financial Tower, No. 02 Hai Trieu Street, Ben Nghe Ward, District 1, Ho Chi Minh City
  • Role in Event: Attendee

Brief Description of Content and Main Activities:

  • Build Second Brain (Tinh): He shared how to build a “second brain” to store, organize, and exploit personal knowledge more effectively in learning and working.
  • Friendly AI Assistant (Hai Anh): Introduced how to build user-friendly chatbots capable of supporting data analysis and visualization on the AWS platform.
  • From Edge to Origin (Thinh Nguyen): Presented methods for optimizing global content delivery using Amazon CloudFront, helping to reduce latency, accelerate access speeds, and safeguard origin servers.
  • 36 hrs with LotusHacks (Team VIB): Brought a practical perspective on teamwork within a Hackathon, spanning from the ideation phase to the continuous 36-hour development and demo of the UTMorpho product.
  • How LLM actually works? (Dao Duc): Helped attendees better understand large language models, particularly the Transformer architecture, Self-Attention mechanism, Pre-training, and Fine-tuning.
  • Enterprise-Grade Multi-Agent System (Cat Vy): Introduced how to design Multi-Agent systems to automate business workflows such as financial analysis and credit scoring.

Results or Achieved Values:

Technical Knowledge and New Technologies:
  • Better understood how large language models work, particularly the roles of Transformers and Self-Attention.
  • Grasped the differences between Pre-training and Fine-tuning when applying LLMs to enterprise problems.
  • Gained more perspectives on how to build Multi-Agent systems to automate complex business workflows.
  • Understood how Amazon CloudFront supports accelerating content delivery from Edge Locations to the Origin while improving web system security.
Soft Skills and Working Methodologies:
  • Learned methods for organizing personal knowledge through the Second Brain framework.
  • Gathered experiences on time management, team coordination, and handling pressure under a Hackathon environment.
  • Recognized the importance of presenting ideas clearly during technology product demos.
Practical Application Values:
  • Able to apply Second Brain thinking into the process of learning AWS and taking workshop notes.
  • Gained further direction on building chatbots to support users and visualize data.
  • Better understood how to design CDNs using CloudFront for future labs or internship projects.

Event 2 - FCAJ Community Day (June 6)

  • Event Name: FCAJ Community Day
  • Event Time: 9:00 AM – 12:00 PM, Saturday, June 6, 2026
  • Location: 26th Floor, Bitexco Financial Tower, No. 02 Hai Trieu Street, Ben Nghe Ward, District 1, Ho Chi Minh City
  • Role in Event: Attendee

Brief Description of Content and Main Activities:

  • Docker Containerization (Bao Huynh): Presented the differences between traditional virtual machines and containers, while guiding how to use Dockerfiles and Docker Compose to package and deploy applications consistently across multiple environments.
  • AWS WAF with Machine Learning (Le Hoang Gia Dai): Introduced how to combine AWS WAF with log data and Machine Learning to detect anomalous behaviors, thereby automating the update of IP blocklists.
  • Multiplayer in the Cloud (Nguyen Quoc Bao): Shared deployment architectures for dedicated game servers on AWS, utilizing services like AWS GameLift or ECS to support real-time connectivity between game clients and servers.
  • Effective Teamwork Methods (Truong Phuoc): Focused on Agile/Scrum methodologies, how to organize daily standups, sprint retrospectives, manage tasks using Jira/Trello, and coordinate source code via Git.
  • AWS Neptune & GraphRAG (Viet Phat): Presented how to use the AWS Neptune graph database to build a Graph Knowledge Base, supporting the enhancement of retrieval capabilities in RAG systems.
  • IT Helpdesk to Senior Sysadmin (Vinh Tran): Shared a career development roadmap climbing from an IT Helpdesk position to a Senior System Administrator, encompassing Linux/Windows Server, Python automation, Terraform, and building a Home Lab.

Results or Achieved Values:

Technical Knowledge and New Technologies:
  • Better understood the process of application containerization using Docker and Docker Compose.
  • Learned how Docker helps resolve environment discrepancy issues between local machines, servers, and deployment environments.
  • Grasped how AWS WAF can combine with log streaming and Machine Learning to enhance defensive capabilities against attacks.
  • Understood the basic architecture of dedicated game servers on the cloud and how to handle real-time connectivity.
  • Acquired further knowledge on GraphRAG, AWS Neptune, and how graph models support multi-step information retrieval.
Soft Skills and Working Methodologies:
  • Better understood how to organize teamwork according to Agile/Scrum.
  • Learned how to break down tasks, monitor progress, and coordinate within a team through Jira, Trello, and Git.
  • Gained more direction on constructing a personal learning roadmap to develop from IT Helpdesk to Sysadmin.
Practical Application Values:
  • Able to apply Docker to package demo applications or workshops during the internship process.
  • Learned how to use Git and task management tools to coordinate better within a team.
  • Acquired further ideas on applying proactive security using AWS WAF and log analysis for cloud projects.

Event 3 - FCAJ Community Day (June 27)

  • Event Name: FCAJ Community Day
  • Event Time: 9:00 AM – 12:00 PM, Saturday, June 27, 2026
  • Location: 26th Floor, Bitexco Financial Tower, No. 02 Hai Trieu Street, Ben Nghe Ward, District 1, Ho Chi Minh City
  • Role in Event: Attendee

Brief Description of Content and Main Activities:

  • Deep Response Engine (Steve Tran): Introduced the mindset transition from an alert-driven operations model to an action-driven operations model, where systems can detect, analyze, and handle incidents toward a self-healing direction.
  • Voice Agents at Scale (Nghia Danh - Trung Vu - Kiet Tran): Presented how to build voice agents utilizing Amazon Bedrock and Amazon Nova Sonic, focusing on optimizing latency, accuracy, and real-time conversational experiences.
  • AWS DevOps Agent (Nguyen Nguyen - Bao Phan): Introduced a virtual DevOps assistant capable of supporting operations engineers, analyzing faults, and proposing or executing incident responses on ECS container environments.
  • Workforce Planning for Enterprise (Truong Tran - Minh Anh): Presented how to apply Amazon Q to analyze HR data, supporting enterprises in resource planning and workforce productivity optimization.
  • Secure Private MCP Connection (Toan Nguyen - Nghi Danh): Shared how to establish private, secure connections between AWS services and Model Context Protocol servers, restricting data transfers over the public internet.

Results or Achieved Values:

Technical Knowledge and New Technologies:
  • Gained more understanding of the development direction from Alert-driven Operations to Action-driven Operations in cloud system management.
  • Grasped the idea of building systems that automatically react and self-heal when incidents occur.
  • Approached new voice agent models utilizing Amazon Bedrock and Amazon Nova Sonic.
  • Understood how AI Agents can support DevOps in detecting, analyzing, and resolving container faults.
  • Acquired further knowledge on using Amazon Q for enterprise data analysis and resource planning.
  • Understood the roles of VPC Endpoints, private connectivity, and MCP in strengthening connection security.
Soft Skills and Working Methodologies:
  • Enhanced system design thinking toward a Security by Design approach.
  • Understood the importance of measuring operational metrics such as MTTD and MTTR.
  • Trained capabilities in observing, analyzing architectures, and evaluating the application value of AI solutions in enterprises.
Practical Application Values:
  • Gained further direction on applying AI Agents into system monitoring and operations support.
  • Understood how to design secure private connections for critical services on AWS.
  • Able to apply knowledge regarding network security and operations automation into cloud security labs or internship projects.
