### Week 7 Objectives:

* Learn how to build GraphQL APIs using AWS AppSync.
* Practice deploying a simple containerized application on Amazon Lightsail.
* Get familiar with the process of building a Data Lake on Amazon S3.
* Learn how to create a Data Catalog and query data using AWS Glue and Amazon Athena.
* Practice building visual dashboards using Amazon QuickSight.
