### Kết quả đạt được tuần 7:

* Hiểu sự khác nhau cơ bản giữa GraphQL API và REST API.
* Tạo được GraphQL API bằng AWS AppSync.
* Biết cách kết nối AppSync với Amazon DynamoDB thông qua Resolver.
* Thực hiện được các thao tác thêm, đọc, cập nhật và xóa dữ liệu.
* Hiểu vai trò của Amazon Lightsail Container trong việc triển khai ứng dụng Container đơn giản.
* Triển khai được ứng dụng từ Public Docker Image trên Lightsail.
* Biết cách build, push và tạo phiên bản Deployment mới cho Container.
* Hiểu các thành phần cơ bản của kiến trúc Data Lake:
  * Amazon S3
  * Kinesis Data Firehose
  * AWS Glue
  * Amazon Athena
  * Amazon QuickSight
* Tạo được S3 Bucket và Delivery Stream để thu thập dữ liệu mẫu.
* Sử dụng AWS Glue Crawler để nhận diện cấu trúc và tạo Data Catalog.
* Thực hiện được các truy vấn SQL cơ bản bằng Amazon Athena.
* Hiểu các thành phần Data Source, Dataset, Analysis, Visual và Dashboard trong QuickSight.
* Tạo được các biểu đồ, KPI, Pivot Table và bộ lọc dữ liệu.
* Xuất bản được Dashboard cơ bản trên Amazon QuickSight.
