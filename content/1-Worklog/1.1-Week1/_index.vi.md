### Kết quả đạt được tuần 1:

* Hiểu tổng quan về AWS Cloud và các chương trình AWS Free Tier.
* Biết cách sử dụng AWS Budgets để theo dõi chi phí và thiết lập cảnh báo qua email.
* Nắm được quy trình tạo và quản lý yêu cầu hỗ trợ trên AWS Support.
* Hiểu các thành phần chính của IAM:
  * IAM User
  * IAM Group
  * IAM Policy
  * IAM Role
* Áp dụng nguyên tắc Least Privilege khi phân quyền truy cập tài nguyên.
* Khởi tạo và sử dụng môi trường phát triển AWS Cloud9.
* Triển khai website tĩnh trên Amazon S3 và truy cập bằng Website Endpoint.
* Hiểu các thành phần mạng cơ bản trong Amazon VPC:
  * CIDR Block
  * Subnet
  * Route Table
  * Internet Gateway
* Tạo và kết nối đến Amazon EC2 Instance bằng SSH.
* Tạo Amazon RDS MySQL và cấu hình Security Group cho phép EC2 kết nối đến cơ sở dữ liệu.
* Hoàn thành nền tảng kiến thức và môi trường AWS cơ bản cho các tuần thực hành tiếp theo.
