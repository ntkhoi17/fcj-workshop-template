### Week 1 Objectives:

* Learn about basic AWS services and AWS Free Tier policies.
* Get familiar with the AWS Management Console, IAM, and authorization mechanisms.
* Practice deploying foundational resources including S3, VPC, EC2, and RDS.
* Learn how to monitor costs and submit support requests on AWS.

### Week 1 Achievements:

* Gained an overall understanding of AWS Cloud and AWS Free Tier programs.
* Learned how to use AWS Budgets to track costs and set up email alerts.
* Grasped the process of creating and managing support requests on AWS Support.
* Understood the core components of IAM:
  * IAM User
  * IAM Group
  * IAM Policy
  * IAM Role
* Applied the Least Privilege principle when authorizing access to resources.
* Initialized and utilized the AWS Cloud9 development environment.
* Deployed a static website on Amazon S3 and accessed it through the Website Endpoint.
* Understood foundational networking components within Amazon VPC:
  * CIDR Block
  * Subnet
  * Route Table
  * Internet Gateway
* Created and connected to an Amazon EC2 Instance via SSH.
* Created an Amazon RDS MySQL instance and configured Security Groups to allow EC2 to connect to the database.
* Completed the foundational knowledge and basic AWS environment readiness for subsequent practice weeks.
